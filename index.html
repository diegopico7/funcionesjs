Funciones ejercicios

Las funciones son porciones de código o conjunto de instrucciones que realizan una taréa específica y puede utilizarse a voluntad y en cualquier momento que necesitemos en nuestro código

Funciones declarativas

Consiste en declarar la función con un nombre y sus parámetros de entrada entre paréntesis

function hola () { codigo }

Funciones anónimas

const hola = function () { codigo }

Funciones flecha

const hola = () => { codigo (inclusive si es 1 linea de codigo no es necesario las llaves)}
